农师傅椒麻鸡

2021-02-09 — 1 min read
#菜谱 

视频地址：https://www.bilibili.com/video/BV1Rk4y1d7ss

  1.  煮鸡
    1. (optional) 焯水
    2. 煮前用盐+花椒粉搓鸡10min
    3. 加姜片、盐、花椒粒，加水没过鸡，加少量料酒，煮45min
      • 如没有焯水，则在此步骤撇去浮沫
    4. 关火，捞出鸡，拆鸡丝
  2. 配菜
    1. 生洋葱切丝，白/黄洋葱，细丝
    2. 香菜切段
    3. 花生/芝麻
  3. 调料汁
    1. 鸡汤一大勺
    2. 生抽
    3. 盐、糖
    4. 胡椒粉
    5. 辣椒油
    6. 花椒油 （3.5 ： 3.6 = 1：1）
  4. 拌匀，吃
