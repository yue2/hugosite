#同人  #同人 

回顾一些礼节性嗑过的cp, 主要是rps，都是我的个人揣测，但会说得很肯定，因为我对自己的揣测比较自信。可能会冒犯其他粉丝朋友，但是没关系估计也没人看，可能只会冒犯到扁老师。; )

《社交网络》双男主 #

说的就是Jesse Eisenberg 和Andrew Garfield两个演员。可以说是谈过，拍戏的时候至少是很暧昧，所以加菲觉得他们是在谈恋爱，但是卷西并没有把这段关系当成恋爱。所以加菲一直以来都想要一个“爱过”的答案，但是卷西恐怕不会给。现在最多算是普通朋友。

周迅/范晓萱 #

谈过。范晓萱是很有才华的音乐人，她的才能和野心远远不止刚出名时的“小魔女”形象。周迅非常敏感、很容易共情他人，这也帮助她成为了一个出色的演员。所以周迅很容易欣赏别人的才华和魅力，很容易爱上任何美好的东西，也很容易爱人；而范晓萱恐怕很少被这样深刻地理解过，所以引为知己。周迅爱过了就去爱下一个了，范晓萱找到下个一个知己了吗？

Taylor Swift/ Karlie Kloss #

谈过。谈了还挺久的。在taylor swift几十段真真假假男男女女的感情里，这应该是比较付出真心的一场关系。模特对她来说是全新的圈子，karlie kloss也爱带她一起玩。后来不知怎么的吵架了吧，我猜测是taylor swift赌气提分手，kk觉得分手就分手，没有按照swift的设想回头挽留，于是散了。但是swift觉得自己明显是在赌气，就这样分手了很不甘心。现在是不咋联系的前任关系。

《疑犯追踪》人物 shaw/root #

官方盖章谈过。应该早就睡过，在某次任务结束之后随缘一睡，后来又打过几次回头炮。期间shaw可能还睡过其他人，root没有。虽然shaw才是精神异常，但是不愿面对内心感情的恐怕是root。她总是半真半假地调戏shaw，但是嘴上越浪心里越怂，她单方面推断shaw认为她们是偶尔上床的同事关系。所以她不愿意承认自己喜欢shaw，因为在root的价值观里她喜欢别人别人不喜欢她，就等于是她输了，但root不可以输。所以后面root为了救shaw整出的操作稍微夸张了点，但是shaw回来表白（基本上是表白吧）时root又太冷静了点。shaw会喜欢她是她不敢想的事情，root这么自信的一个疯女人，唯独不敢相信自己会被人喜欢。
