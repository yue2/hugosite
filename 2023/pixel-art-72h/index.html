我在三月份的时候说想玩像素画，小半年过去了，一共只画了71.3h，但是因为72听起来比较整，所以起了这么个题目，来分享一下我们三分钟热度也会有的三分钟经验。

怎么开始 #

起因是这样：

我上学期参加了一个拼豆小活动，做了一个小挂件。我一边做一边就想，这不就是传说中的 pixel art, 只是做成了实体，也就是豆。那我既然能拼豆，也能数码拼豆吧？——即为像素画。我觉得这事儿可行，于是上网征求网友意见：

网友们推荐了很多，其中最重要的我认为只有提到软件的这一条，那就是大家都用 aseprite, 而且这个软件在 steam 上购买比较便宜。

选择软件 #

aseprite 在steam上其实也没有很便宜，折合人民币一百块。我当时刚搬家一个人住并且为此付出了非常贵的房租，所以虽然一百块这个绝对值不算高，但是还根本不知道自己会不会用呢，就要花一百块买个画画软件，这件事给我的心理压力非常大。

现在看来当然是很值的，毕竟一百块的游戏可能都玩不了70h。这个软件是专业选手也会用的，所以在功能上不用担心有任何别人能做你做不了的。大多数能找到的网上视频也都是用这个软件画的，所以也比较容易学。

如果想用免费的，在ipad上也有一些，功能简单很多，就是一格一格上色这样的画法。当然了，理论上讲画像素画并不需要任何特殊软件，在电脑自带的画图上，把笔刷设置成方块儿，画出来的自然也是像素画。但是这个对于新手来说一点帮助也没有，就像人家说想学写代码，你说：“想写就能写！打开记事本立刻开写！”这是非常不负责的推荐。

专门的软件，在这里说的就是aseprite, 有很多实用功能，比如说自动去重合线（anti-liasing）、缩略视图、预设色盘、自动加边框等等，老少咸宜、事半功倍！但是这个软件是设计给鼠标+键盘/手绘板使用的，我也试过用 steam link 到 ipad 上画，操作起来非常难用，完全不推荐！

这个艺术形式的优点 #

像素画这个形式，最大的优点就是成本低。你可能要说，这不是才花了一百块买个专门的画图软件？——是的，但是它对电脑的硬件要求非常低，毕竟这只有几百个像素。我还玩过不要钱的软件blender, 但是没有独立显卡，这个甜甜圈渲染了我二十多分钟：

而实体的画画又要买耗材，画出来的画还得有地方放，约等于还得买个房。所以不管怎么算，像素画都是非常低成本的活动。

那像素画的另一大优点就是门槛低，也就是说不需要什么美术基础，光影、色彩在像素画里都是被高度概括的，而且因为像素的限制，复杂程度低了很多，还有很多人会分享色盘给大家参考（直接抄）。

在哪里可以学到 #

我用到的学习资源主要都是youtube, 总结一下我看过的博主。以我的冲浪经验来看，像素画作画是非常小众的，大家做像素画主要是为了做游戏。这些博主大部分是游戏开发者，和我的目标并不一致，我只是想画画，而且我平常也不爱玩游戏，所以他们说的很多都跟我没关系，我就会跳着看。

pixel art 101 #

这个是很多人都推荐的系列，但是我并不太喜欢，因为这个人没有按照任何规律来安排教程，除了前几个教怎么用软件的，后面都是以画了什么 object 来安排的视频，对新手来说有点东一榔头西一棒槌。这个博主作为艺术家和开发者，技术是很好的，也有很多 insights, 但是我觉得这个教学水平不行。

——不好意思我们小聪明就是会 judge 老师的。

但是他有一些小撇步说的很好很有用，其中最重要的就是打开一个 pinterest 收藏板，把你看到的所有喜欢的像素画都放进去，不拘风格形式，只要是你喜欢的想参考的都加上。如果你用 twitter, 还可以添加一个关注列表，把像素画艺术家都加进去，有事没事刷一刷，培养自己的品味。

pixel art class by AdamCYounis #

这个视频列表我觉得更合适，博主讲得很慢很细，所以每个视频都挺长，可以1.2倍速观看。他会解释整个创作流程，从 freehand 草图开始，到每一步的细化，不会让人感觉开场面对空白画布很迷茫。但是这位博主也是游戏开发者，而且目前正在一生悬命，专职 solo dev 命中注定的作品，所以他的教程列表里大部分是和游戏场景更相关的图，还有很多动画教程。当然了对新手来说学啥都是学，所以也不错。

pixel art tutorials by Saultoons #

这个光头佬的视频比较简短，很容易看。光头佬也在开发一个小游戏，他的开发日志做得很幽默，我还挺爱看。他每年会发布一个30天挑战，给大家提供画画的主题。这个很有用，因为题目很简单，限制也很多，是很容易上手的练习，不用担心不知道画什么。我就画了一些这个 1 bit 小动物主题：

很可爱，只有16x16个像素那么大，而且只有黑色和白色，但是很神奇，你在画之前不会感觉到256个像素点能放下这么多信息，而你真的画出来就会发现，这些动物甚至能有表情有性格！而且这个狗，它不光是个狗，他还是个比格！

像素画的成就感来得就是这么快，你觉得只是闲暇时光点点鼠标，居然就能画出像模像样的简笔画。

getting started with pixel art #

还有一个我喜欢的博主是 Brandon Greer, 他的教程更简短，有点太短了，不适合跟着学习，但是他还有很多创作过程的 time lapse, 我很爱看，可以看到他的工作流程。比如说他喜欢先用粗笔刷打草稿画出剪影轮廓，前面提到的Adam也是这样操作的。因为他们认为像素画是高度概括的表现形式，那就应该让特征非常明显，光看轮廓也能看出是什么。

猴哥（I mean Brandon）喜欢用 photoshop 画图，他的风格和配色都很有意思，我也挺喜欢。

动画 #

做动画是另一门专业，但是像素画非常容易做成动画，而且会动！多有意思！前面几位的像素画教程里多少也有提到，但是我也看了一些动画的基本原理，其中最有用的是这个：

这可是 Genndy Tartakovsky! 传奇动画家！代表作有：飞天小女警等。别的就不说了，飞天小女警就是最重要的 ╰(°▽°)╯

他在这个问答里提到了动画的几个基本原则，像是变形和 anticipation, 只要用上就能让动画像模像样，一种点石成铁的作用——想做成金子还是需要更多水平。

技与道 #

还有这种完全不勾线，几乎是印象派的

还有这样硬勾线的、非常经典动画风格的

因此不必拘泥于任何风格，可以多多欣赏多多收藏。就像前面提到的，我把这些都收藏在 pinterest 里，也不分类也不总结，看到喜欢的就点收藏，然后想起来就看一看。我还会收藏游戏截图，赛博酒吧va-11就是作画很优美的一个像素风游戏。

除了像素画的参考，我也会针对想画的东西收藏很多其它形式的参考，比如说三次元照片、动画场景etc

主要靠观众脑补 #

本文前面也提到很多次了，像素画是一个高度概括的艺术形式，哪怕是这样

你也能认出这都是什么东西。

靠的就是观众朋友们的脑补。因而创作者的一部分工作，就是 manipulate 观众的脑补。这部分我还没学会，只知道是这么回事儿。所以一方面像素画好像每个像素的位置都很重要；另一方面好像一整块一整块的色块，只要排布成差不多的形状，看起来都挺像样。

做一个项目——我学任何东西的方法 #

我在上手了基本操作之后，就会想做一个项目。这是我学习任何东西必经的道路，就是会想象我要用这个工具来做什么，然后做自己的项目经理，详细提出一个方案。如果教程就是 project-oriented 那最好，比如说那个 blender 甜甜圈教程；如果不是，那我必然半路脱缰，开始操作！而且做一个小项目必然会用到入门教程没有教的内容，我就会一边做一边谷歌，一边谷歌再一边做。所以我很难完成一个教程，因此也很难扎实地学完任何东西。

但是我已经和自己和解了，我们工程师就是这样的，能用就行，一切都是为了解决问题！

形式与媒介变化多端但创造力来自脑子 #

形式是内容的一部分，形式可以影响内容创作的思路，但是产生创作的还是创作者的脑子。像素画在很大程度上减轻了技术要求，但是想要不断进步还是很难的。而创造力又是不会被形式束缚的，我觉得好的作品都是利用表现形式做出的表达，是互相选择而结合的产物。

当然了，我对自己这种业余涂鸦没有什么要求，但是我在欣赏其他作品的时候是会这样看的。

个人感想 #

对我来说，这像素画72h里的阵痛主要来自于眼高手低，也就是我的审美趣味和实现水平之间的巨大落差。理智上很好理解，就是我作为一个观众已经在鉴赏水平上训练了太多，而我从来没有画过画，所以水平很菜。

另一方面因为我平常喜欢拍照，所以对光影明暗关系和颜色有一点了解，感觉学起来很容易上手。我同时也感觉到了这给我带来的限制。因为我对色彩的了解几乎完全来自三次元，我很难想到去用“不现实”的颜色，而这正是像素画的一大魅力。所以我也觉得学到很多，很有意思。

我自己是一个把创造，而不是消费，作为休闲娱乐的人。就是说我闲下来也会看书看电视，但是这种纯粹消费行为会让我觉得无聊，要解决无聊我会需要做出点什么。这两种方式当然没有高下之分，因为我也只是在制造垃圾。你可能看到成品觉得我画得也还行啊，像模像样的——那是因为我真的很聪明，学啥 tutorial 都能整个像模像样的。我是就凭借这点混到了现在，但是每当我想更进一步，就会感觉到阻力很大，于是又会看上另一个活动，开始下一个三分钟的热度。没关系，啥都学点又不是什么坏事。

总结来说，像素画是非常低成本的休闲娱乐，能很快获得成就感，制造出来的垃圾也不占地方，非常适合我。
