有的东西用了很久

我觉得一款产品（不是车不是房）能用五年差不多是很久了，此刻是2023年，五年前就是2018，因此本题也可以说成是：看看我在2018年左右买的哪些东西到现在还在用呢？

我其实是很懒一个人，软件更新都不积极，所以一个东西用五年不更新迭代对我来说也没有非常罕见。如果一个东西的功能足够基础、产品又足够可靠、科技上没有划时代的更新、外观上没有很夸张不会一眼就看得出过时，我不会觉得有换掉它的动机，基本上都是一直用下去或者无限回购。

本文提到的东西将涵盖乱七八糟各种品类，如果有读者朋友觉得很过时了，也欢迎推荐新一代产品！

前提都说完了，那么就开始吧！

Bialetti 摩卡壶 #

现在这只摩卡壶是我2018年买的，来自意大利的Bialetti. 这个牌子基本上是摩卡壶的同义词，只要看到它六棱台的造型和标志性的小人儿指天Logo应该都能认出来。买它的契机是我上大学的时候有一年去欧洲交换，宿舍里有个意大利室友。这个室友很爱做饭，也煮咖啡，用的就是一个摩卡壶。

我和同学问人家：这个壶看起来挺好用，哪里买的？

室友面露难色：我从罗马背来的。

此番对话给我留下了深刻的印象。一个咖啡壶好用到要从家里背着来上学，是怎样优秀的品质！于是我也给自己买了个。我精心挑选了一个最小的1杯装最基础款，从淘宝买的，好像是一百多块人民币，带着它开始了本人的喝咖啡生活。

摩卡壶整体是铝制的，把手是塑料的，一杯装的很小一个，确实很容易带，我也背着它上了很多学。这个东西煮咖啡的原理是完全机械的，把水加热烧开，水蒸气自己加压通过咖啡粉，萃取出咖啡。所以它没有任何附加配件，只要有加热台子就能用，煤气灶、电热灶都可以，近些年流行的城里人野营场面里也经常出现摩卡壶，就是放在露营瓦斯炉上用的。也非常适合我这样居无定所的年轻人，每年都会搬家，拎起来就跑。

摩卡壶煮咖啡比espresso的压强（9 bar）略小，但是香气和油脂都能保留一部分，早上起来操作一下也很方便，将摩卡壶坐上灶台，就可以往杯子里加牛奶和冰块，然后咖啡煮好离火倒进杯子里——获得一杯dirty.

  • 使用频率：高
  • 推荐指数：高
  • 耐用程度：高 ——这个东西除了看起来脏了点完全没有要折损的意思，是传家宝程度的耐用！

keepcup 咖啡杯 #

来都来了就顺便说一下这个咖啡杯，这是2018年小扁送我的生日礼物。当时应该是看我开始喝咖啡了吧，也可能是我自己想要的。我对比较熟的朋友都会直说我想要什么礼物，就是分享我的wishlist让他们自己从中挑一个送我。我觉得这样比较好，我不会收到不想要的东西，朋友们也会知道他们送我的东西我会经常用。另一方面，对于不是那么熟的朋友，我希望他们送我一些比较随机、我几乎完全不懂、没有偏好的东西。我经常很怕不是那么熟的朋友听说我喜欢喝咖啡，特地给我送了两包速溶……

言归正传，这只keepcup. 我想自己带杯子装咖啡环保一点，这个长得也比较好看，中间一圈防烫手/冰手的软木环设计的也很合理，所以想买。作为礼物，颜色是扁扁挑的，选了原色软木环配灰色白色的盖儿。完全就是我喜欢的那种精心挑选后和默认款长得一毛一样的东西！

但是实际使用起来，这个杯盖儿并不防漏，不能把这个杯子扔进书包里一阵狂奔去追校车。那你叫什么随行杯啊！嗯？？

接下来就很快转职成了在家喝水的杯子，这个任务完成得相当好，它用的厚玻璃做杯体，失手从桌子上砸下去是不会碎的，软木环也很好握。

  • 使用频率：高
  • 推荐指数：普
  • 耐用程度：高 ——脏了就用海绵蘸牙膏刷一下，可以用到地老天荒

飞利浦电动牙刷 #

最便宜的那款，二百多块的。我大概就是从2017/2018开始使用电动牙刷，一直是这款最便宜的飞利浦，但是它坏过两次，所以这已经是第三支。电动牙刷么，有这个电动的功能差不多就可以了，这个两分钟计时也是我非常需要的，除此之外的功能我都不需要。电动牙刷的高附加值功能都很让我无语，无线充电可能还有点道理，要下载手机app就是我绝对不能忍的！我宁愿多付钱买不要我下载app的东西好吧！

这个牙刷好用是好用，简朴是简朴，但是每到两年的寿命时都会自己坏掉，很神秘，我已经经历了两次。

第一个是无法充电了，我当时还很伤心，因为我是一个挺珍惜物品的人，无缘无故坏了我就希望能修好。但是两年已经超过保修期了，而且这种产品，内部元件一般集成度很高，基本上没法拆开修。我伤心，于是小扁提出再给我买一个，又买了个一毛一样的。

又两年过去，第二个也坏了，还坏得很灵异，有一天我在桌前，突然听到它在卫生间自己振动起来了！我发现握柄上链接刷头的部分那个橡胶圈被腐蚀变形了，经过一系列谷歌搜索，我诊断是这个橡胶圈腐蚀后不再防水，进而水进到握柄部分的电路，造成了短路，牙刷自己就不受控制地开开停停。

总之现在就买了第三个，继续用着，我已经基本承认了两年就要换一把的期待，觉得也还可以接受。

  • 使用频率：高
  • 推荐指数：高 ——大品牌，经典款，功能齐全，也不贵
  • 耐用程度：普 ——两年换一次

科颜氏高保湿霜 #

大宝天天见！最近经常在小红书上看到用了一两年的选手说自己是长期用户来教大家怎么用，那我不得不说，一两年算什么长期？这个在上一集消费主义陷阱节目中就说过了，再抄一下：

这个不是2022年才买的，我每年都买，追溯起来是高中同学给我推荐的产品，我不知道是上高中还是刚上大学就在用。这么多年了，我妈都觉得我该开始抗皱了，我才刚刚知道它的好！这个霜，我和对象内部称之为大宝，因为小时候经常放的“大宝天天见”广告深入人心，而在我们心目中科颜氏高保湿霜就是一切大宝希望自己能成为但是没有做到的——当然啦，是因为这个比大宝本宝贵多了。主要是它平平无奇的白罐盒子里平平无奇的白色面霜，每天平平无奇地抠出一坨糊在脸上，就好了！是的，我的护肤流程一般来说就是这样了，想得起来就多涂一个水，常用的是悦木之源蘑菇水、科颜氏金盏花、科颜氏高保湿白瓶那个水。 我的肤质称不上好，这个护肤流程也没多少值得借鉴的。但是我的脸算是比较敏感，换产品很容易脸颊泛红，严重的时候会出小疹子，这也是这么多年没换产品的一个原因。刚刚过去的寒假里我先涂了几天我妈的雅诗兰黛，整个脸都痛，脸颊上直接干到脱皮了，赶紧买了罐大宝，糊了三五天就好了。真的是往脸上糊，涂上比平常用量多的面霜，隔段时间觉得脸颊干了就再往这个区域再涂一点，就好了。

这个脸霜的功能简单、效果显著，我没有更多要求了。我妈认为我该开始抗皱了，我可能在大宝的基础上再买个精华之类的涂一涂就是了。

  • 使用频率：高
  • 推荐指数：高
  • 耐用程度：低 ——消耗品

希捷移动硬盘 slim #

我买的这个500GB，上高中的时候买的，算下来快十年了。去年扫过没有坏道，读写速度确实是上个时代的产品，但是读写都很安静，而我后来买的一个东芝随便用用就转得呼呼响！这个希捷现在还在用，非常方便。

  • 使用频率：普
  • 推荐指数：高
  • 耐用程度：高

kindle paperwhite 2 #

2014年买的，还活着，还能用，但是使用频率急剧下跌，最近几年已经开不了几次机了。更无语的是，我的中国亚马逊账号绑定的是163邮箱，这个邮箱当时没有绑定手机号，反正就是完全登不上去了！kindle商城也要关掉了，我又没法换区出来，现在只能看原来有的和用数据线导进去的盗版书。

  • 使用频率：很低
  • 推荐指数：普
  • 耐用程度：高 ——在电子产品里出类拔萃

remembear 密码软件 #

这个密码软件我大概在2016年左右开始用，第一年是beta, 之后每年36美刀全平台同步，不付钱的话就只能本地用不能同步。在当时密码管理软件的需求刚开始铺展开，我需要一个跨很多平台、方便好用的密码软件，就选中了这一个，它的美工也很可爱，是个熊。有pc端、ios/安卓app、浏览器插件，可以自己录入密码条目，也可以从浏览器中导入，还可以生成随机密码。

该有的功能都有了，界面简洁，功能稳定，没有出过差错，所以我根本没有更换它的理由。这种安全类产品或许就是这样的，一开始获得了用户，粘性就会非常大。

但是它要倒了！ 我又把东西用倒闭了！这个软件将在2023年6月停止服务，最后一年也不收钱了。让人伤心！但是这几年间密码管理软件越来越多做得越来越好，找一个合适的替代品应该不会很麻烦。

  • 使用频率：高
  • 推荐指数：低——即将关停服务
  • 耐用程度：高

其它 #

除此之外还有几件五六年前买的衣服我现在还在穿，主要是一些外套：一件冲锋衣、一件牛仔外套、一件短棉袄。每年穿着频率不是很高了，但是确实还在衣柜里，每年都会穿一穿。

总结下来，针对基础功能，买的时候选择比较贵、质量比较好的产品，没有意外都可以用很久。它们不需要表现很好，只要表现不是很差，能够满足我最开始的期待，都能获得多次回购！
