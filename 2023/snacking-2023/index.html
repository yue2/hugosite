盘盘最近爱吃什么零食！最近指的是2022-2023吧。

本来想插入 amazon 链接方便大家看的，结果发现 amazon 要求我注册成为 affiliate 才能分享图文链接，怪麻烦的，这笔抽成我先不赚了，你们自己看名字去搜吧一样的！

咸口 #

蘑菇干 #

我买的这个牌子叫 vegky, 意思就是 vegetable jerky 呗，蔬菜做的牛肉干。这个是用蘑菇做的，吃起来像是杏鲍菇类似的蘑菇，质感和调味都做成类似牛肉干的样子。我喜欢吃这个已经超过了牛肉干，因为我有时候不想吃那个牛味儿，只想吃一个咸鲜口的零食。而且有的牌子的牛肉干嚼劲也太疯狂，偶尔会担心下巴脱臼！这个蘑菇干比较软，吃起来不费劲。好像有很多竞品品牌，但是我还没吃过，所以只能分享这个。这个牌子有好几种味道，我最喜欢的是照烧味（teriyaki），这个辣味不好吃，没有香辣感，只有干辣，一种白人瞎撒辣椒粉的感觉。我自己不是 vegetarian 或者 vegan, 所以我不清楚配料里有没有动物制品，有饮食习惯的朋友可以自己检查下。

海底捞小锅巴 #

没错……就是海底捞门口等位会送的小锅巴，还有虎牙脆，约等于是迷你版的妙脆角。这玩意儿吃起来就是一口里有半口味精，怎么可能不好吃？网购可以买到，不用去海底捞偷。 这个一包非常小，两口就没了。有时候我这个零食瘾恰好就是这两口的事儿，所以吃完一小包就不吃了，也很合适。

卡乐比薯条 #

大名鼎鼎的 Jagabee 如果还有朋友没吃过请一定吃一下！真的太好吃了，这个薯条干干脆脆，麦当劳都比它油腻。普通薯片囿于形状，很容易觉得就尝个粉的味儿，土豆并不多，薯条就很完美，大部分还是土豆的口感和味道，调味也不是很咸。我最喜欢的是海苔味的薯条，其次是盐味的紫薯条，第三名是盐味的薯条。缺点是贵 ，卡乐比在零食界还是很高端的，至少定价上是这样，如果不常吃零食那偶尔花点钱也还可以接受。薯条也是一小包一小包的，一次吃一包解个闷正好。卡乐比还有个荷兰豆，也是酥脆的咸味零食，也很好吃，其中芥末口味的格外好吃。但是荷兰豆在本地铺货不多的样子，所以我不常吃。

苏打饼干 #

永远经典的苏打饼干我永远的爱。上中学时我就喜欢带去学校吃，会买太平苏打和四海苏打，太平的香葱味很好吃，四海我喜欢番茄味和海苔味的。现在容易买到的是明治和嘉顿，这两个牌子我都喜欢芝麻味。我出门都会带一小包在书包里，以防不时之需，就是突然好饿就掏出来吃一吃。其实以我这样的性格，备用方案从 B 排到 L/M/N，很少有吃到这包备用苏打饼干的机会，反而是一出门就拿出来接济同行朋友的时候多。如果没被毫无计划的朋友吃了，那几个月之后我就会从书包底端掏出碎得不成样子的饼干倒进碗里舀着吃完。

甜口 #

莱布尼茨动物饼干 #

莱布尼茨真的很好吃，原味就是香香甜甜的黄油味道，燕麦味还有一点点燕麦的质感，我非常喜欢。这个饼干口味就是清清爽爽的、相当 generic 的饼干味道。大概就是柏拉图“理想的马”那个说法，就是说我们脑子里会有一个马应该有的样子，现实中其实没有哪一匹真正的马是和你脑子里这个理想的马是一模一样的。现在在脑子里想象一个微甜的黄油饼干——莱布尼茨就是这个味道和口感。动物饼干还做得很可爱，网友认为打入中国市场时应该叫“羊顿”。

我这周又试图做个大人，买了长方形的普通莱布尼茨。

明治杏仁巧克力 #

我觉得精神不好感觉身体不舒服的时候吃点甜的会好一点点，所以对自己吃巧克力没有很限制，其中我比较喜欢的就是明治的杏仁巧克力。就是在一整颗杏仁外面裹上相当厚的一层巧克力，明治有出牛奶巧克力和黑巧克力的选项，我喜欢吃黑巧克力的，大概 65% 可可含量，不会觉得太甜。

ORIHIRO 零卡果冻 #

这个也是网友推荐的产品。我挺喜欢吃果冻的，但是长大了以后因为不想吃糖所以很久很久没吃过了，这个零卡的产品当然就特别适合我。说是零卡，实际上当然用到了代糖，健康风险大家自己平衡。这个牌子出很多水果味，味道还比较真实，但是代糖零卡嘛，大家也不要期待太多。我最喜欢的是巨峰葡萄味，比其他虚假的葡萄味产品要好吃一些。这个也是一小个一小个单独包装的，有效控制摄入量。我喜欢把果冻冷冻起来，然后拿出来等它稍微化一点的时候吃，外层是果冻口感，内层还像吃冰沙，非常爽！这种独特魅力就类似于绿舌头。

荣誉提名 #

这个组别是我认为还是好吃的但是已经不常买了的产品。

嘉顿香葱饼干 #

上面说到了太平牌香葱苏打，但是最好吃的葱味儿饼干一定还是嘉顿香葱饼干！这个我认为很有说服力，因为我甚至不吃葱。我不吃新鲜的葱，但是葱油我是吃的，所以我也喜欢吃葱油拌面和葱油饼。这个嘉顿的香葱饼干就是葱油饼味儿，和我外婆会做的葱油饼味道很像，而质地就像一整块都是葱油饼最边边上焦脆的一圈。这是它的魅力同时也是缺点——吃多了很油，一块儿又特别大，中学之后我就没那么喜欢了。

Kellog’s super K 早餐麦片草莓味 #

里面有真的冻干草莓。可能我就是喜欢吃冻干草莓但是直接买太贵了。
