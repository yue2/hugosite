你需要的是这个运动手环吗？反正我不是

2023已经走到11月，今年的盘点就从一些比较失败的事情开始吧！失败是成功之母！让我们吸取教训越买越好！少买没用的，多买有用的！这个华为运动手环5是前两年就买了的，但一直处于鸡肋的状态，用得不多，但又不是不能用，因此弃之可惜。

怎么买了 #

我一开始决定买运动手环，就是想记录自己的运动。这并不是什么新鲜事，甚至我可能已经落后潮流将近十年了。考虑买手环而不是智能手表的原因也很简单，我不想要手表的功能，我只想要运动记录的功能，这个设备我只会在运动的时候用。而且运动会出很多汗，表带浸了汗会很恶心，我不想运动完还戴着，所以更不想要主打24h的智能手表了。

我买的是华为 band 5，是功能最简单的旧款。华为这个手环不贵，人民币一百上下，和两千多块的智能手表相比，是不需要太纠结就能下单的产品。所以就买了！

和需求不符 #

使用率这么低的一大原因是它的功能和我的需求并不太符合。我用下来的理解是，一个手环核心功能就是追踪位置和记录时间，所以它适合的运动就是在时间空间上变化的运动。这是在说什么呢？——就是跑步、骑自行车、游泳这些运动——都是我不太喜欢的运动。跑步和骑车是我根本不搞的活动，而手环刚买回来的那一阵子我还很经常游泳，所以用得比较多，能体会到一些优点。比如说这个手环可以自动检测泳姿和转身，还会帮你算那个平均划手效率的数据。如果是比较喜欢这几项运动的朋友，或许这个手环会适合你。

我平常的运动习惯完全不是这样。我每周大约健身房两次，羽毛球场两次，都不是这个手环内置的功能。内置运动除了上面提到这几项，还有跑步机和椭圆仪，总之都是有氧心肺锻炼，所以适合我的就只有free training这个选项，就是它开始计时并且监测心率，再根据我的心率算一个卡路里消耗，别的也帮不上忙了。

这个功能到这儿也算是有用吧，我其实也就只想知道这些，更复杂的数据有没有都行。但是很明显，这个设备不是设计给我的。

操作不顺手 #

更大的问题出在了UI和UX考虑上。

手环的界面交互特别不好用，要翻好几下、按好多东西才能开始记录，我要用的free training功能又在最后一个，它也不能设置快捷开始，每次都要翻好几下按好多东西！我更需要用的 timer 和 stop watch 都在菜单的“更多”里，又要翻更多下、按更多东西！

这个手机软件也是垃圾！华为health，什么垃圾东西！把广告活动放在非常显眼的地方，个人数据显示这么简陋，统计功能基本没有，只能导出个图！就这，就这还要我注册账号登录，有必要吗？嗯？？

其他功能更不需要 #

消息提醒 #

我的手机是三星的，原装安卓框架，我也不在中国生活，因此非常不想让华为产品获得很多手机权限，所以我根本没开通软件消息通知功能，只开了来电提醒。只开放这点权限是我自己选的，效果也不差，我只是想要在锻炼的时候如果真的有什么要给我打电话的急事我能赶上，但是我确实主动阉割了它能实现的更多功能。

睡眠监测 #

能用，至于准不准确我倾向于认为不是很准但也不差，拿来追踪一个趋势是非常合理的。问题出在哪儿呢？出在了它要给我的睡眠打分。我看到这个分数就有一种做题家的条件反射，那我必须反思！必须进步啊！——但睡觉不是这么一回事儿，睡前越是想着要好好睡就越睡不好，徒增焦虑，至少我是这样。

也不是一无是处 #

这个手环的续航很不错，我感觉能有一个礼拜，就是我每次准备去锻炼，捞起来都还有点电。就算快没电了，充一会儿就够用了，不像智能手表每天都要充。这也是我买手环而不是手表的重要原因之一，我不需要更多高级功能，于是不需要功能更强能耗更大的硬件设备，于是不需要经常给它充电。

那我想要什么样的产品 #

心跳监测是我需要的、睡眠监测有也可以、卡路里计算马马虎虎就行，我最想要的是秒表、倒计时、和计数器。而且我想要实体按钮，我作为一个工程师，就是喜欢实体按钮，别有一种安全感，而且在小屏幕上摸来摸去真的很烦！现在这个手环屏幕上如果有水，触摸屏还会没有反应，更烦了！这样设计也很愚蠢，你既然是一个运动手环，滴汗和游泳池的水都是特别常见的情况吧，这样都会误触和没反应，那你要怎样？软件功能上我理想中应该可以提前输入一些常用的倒计时和计数，比如说三十秒、一分钟、做12个rep，我就可以快捷开始这几个倒计时/计数。然后它可以根据动作传感器知道我做了几次动作，提醒我还有最后三、二、一——完成！

那么我心目中的这个产品就会记录下一些时间和次数的数字，以及我当时的心率。然后我可以在手机或者网页软件里把每段数据对应到我的锻炼计划上，我就可以知道今天的表现怎么样、每个重量做了几次、打球到什么时候会心率太高跟不上场上节奏。除此之外我还是不想要运动以外的任何功能，不要接电话打电话。如果一定要有，那就只要切歌功能，但也不是必需的，因为我的耳机有按钮。

迄今还没找到满足这些需求的设备，欢迎您推荐！
