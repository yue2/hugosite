上次去马来西亚旅游，目的地是东部砂拉越州的首府古晋市，我觉得还挺有意思的，但是别的去过的同学觉得小地方破破烂烂有啥好玩的。如果你也不喜欢破破烂烂的小地方，那你可能也不会喜欢，如果跟我一样就喜欢瞎逛逛那可能你也会喜欢。

多图少字！

地理知识 #

马来西亚领土分布在两个岛上，中间隔了很大一片海，简单来说就叫做东马和西马。西边是一个半岛，是泰国湾延伸下来的那部分，和泰国接壤。几个大城市，比如吉隆坡和槟城都在西马。东边这一整个岛叫婆罗洲，北部是马来西亚的领地，文莱也在这块，被马来西亚领土包围起来。婆罗洲的南边大部分的地盘是属于印度尼西亚的，印尼好像叫这个岛加里曼丹。

这次旅游去的古晋市就在婆罗洲的西北边，从飞机上看下去是这样：

河流蜿蜒向北汇入海，很漂亮！

城市风光 #

古晋市非常小，主街道感觉只有一两条，很窄的街两头都是这样东南亚风格的小楼：

临街是小店铺，二楼开窗住人。车辆不多，路边停着的比路上开着的多很多，当地人出行可能还是用摩托车更多。

图1背景里这个金碧辉煌的顶是婆罗洲文化博物馆，本文提到的文化历史知识全是从那里现学的，可能有学错！这个博物馆很值得一去，知识介绍得非常详尽。票价比较贵，是全古晋市唯一物价赶上国际标准的场所。不得不说布展和导览并不是很好，但是婆罗洲的历史确实太复杂了，我从未了解过，还是学到了很多！

这个城市又叫猫城，因为kuching就是猫猫的意思。传说是因为英国殖民者来的时候，问本地向导这叫什么，人大总督手指的是这个城市，而向导以为他在指街上的猫猫，就说这叫猫猫啊！古晋因此得名。

真是很无聊的历史传说呢！但是因此古晋有很多猫猫雕塑，窨井盖上浮雕也是一只猫。

马来西亚有很多民族，华人大概占了人口的20%，古晋就是一个华人城市。我看《流俗地》 学的，马来西亚的学校教学语言有华文的也有英文的和马来文的，古晋城里唯一的高中看起来就是华文学校，看起来非常亲切，跟我家乡镇上的学校看起来特别像，这栋教学楼甚至叫“明德”。“大学之道在明明德”，多少学校抄了这句！谁这辈子没上过两家有“明德楼”的学校！

历史小故事 #

马来西亚是一个联邦 #

马来西亚是一个联邦制的国家，东马的部分，也就是婆罗洲岛被马来西亚占领的这部分，分为砂拉越和沙巴两个省，沙巴在整个岛的最北端。婆罗洲在二战前被英国殖民，在战时被日据过一段时间，最后是澳大利亚皇家海军来解放的。

然后对这片土地上的人来说，要建立一个怎样的社会就完全是全新的问题。他们经历过了殖民统治和法西斯占领，但是第一次要为自己回答，想要一个怎样的社会。马来西亚在1957年独立的时候只有西边，东边这两个州在1963年才加入。在60年代，马来西亚也经历了红色浪潮，就是共产主义者想夺权，在广袤的热带雨林里打游击，让当时的政府很头疼。当然啦，现在我们看到的历史都是胜利者写的，所以就是击退了赤匪，保护了联邦，现在还是一个民主选举的社会。

博物馆里讲的当然是官方宣传叙事，所以也吹了很多联邦政府。但实际上古晋街头建筑通常会挂两面旗，一边是马来西亚国旗，一边是砂拉越州旗，路边广告牌也经常写“更了解东马的品牌！”，我看你们和联邦政府关系很一般嘛！

大航线上的重要一站 #

在英国人来之前，古晋曾经是在文莱王国的管辖之下的，而这条海岸线在大航线上是很重要的一站。丝绸布匹、瓷器香料，从广州和泉州一路向南，到菲律宾的马尼拉，然后就到文莱，再接着过马六甲海峡就进印度洋了，之后就是绕好望角再北上。所以南来北往，此地和泉州感觉还很有点相似之处，看看这一张是古晋市里的庙：

原住民族 #

在更早之前，婆罗洲有很多种原住民族，具体有啥我忘记了！大概是分成靠山吃山靠水吃水的几种，就是在海岸线上捕鱼的人们、在低地种植的人们、还有在山间高地狩猎的人们。博物馆里有很多各个民族的编织品、祭祀品，还有一些信萨满的，陈列里还有萨满医生的药箱。

在这个岛上犀鸟是很重要的一个东西，一种图腾。犀鸟的喙曾经是当作货币流通的，部落的木雕面具也会模仿这个，海边还有一个犀鸟大雕塑：

吃好喝好 #

马来西亚币一个令吉兑人民币一块五，古晋在马来西亚物价也不算高的，吃饭大概是上海的六折，人民币十块钱能吃上鸡饭，加两块就是鸡腿饭。除了海南鸡饭那种白切鸡，还有一种炸鸡的鸡饭，很香，比白鸡饭更有风味。应该是整只鸡炸了再切的，面衣薄，调味挺重的，肉就没什么调味，就是很多汁的炸鸡。和白切鸡一样蘸葱姜辣椒水。

本地的华人基本上都是潮汕人，所以吃饭约等于是潮州菜，街边大排档的炒果条都很香！砂拉越知名特产是哥罗面，但我没觉得特别好吃，就是一种香菇肉燥拌面。

吃海鲜我们去的这里：

应该是一个游客爱去场所，没问题，咱就是游客！比城里其他地方稍微贵一点，但是还是比较便宜的。这是一个半露天的大排档，去每个档口点菜，先选一个海鲜产品，然后选择烹调方法，有很多种做法，我觉得白煮鱿鱼切条然后凉拌成酸辣口的很好吃，酱炒蛤蜊也不错，蛤蜊还挺新鲜的。另外点的炒蔬菜也很新鲜。

夜宵吃了一家串串店，自己拿串儿然后可以选水煮或者油炸，油炸就是把你拿的串儿进面糊里滚一圈然后淅淅沥沥地进油锅，一种不太讲究的天妇罗。煮完或者炸完的串儿再蘸酱吃，有一个甜口的酱油膏，还有一种辣酱，都可以自己拿。吃到比较特别的串儿是地瓜叶子卷空心菜杆，这可真是天才的发明，地瓜叶子表面滑滑的味道很特别，但地瓜杆特别老完全嚼不烂，拿地瓜叶来卷空心菜杆子，这就是完美的！还有一个鱼饼卷金针菇，就跟什么培根、肉片、豆皮卷金针菇都一样，只是换成了鱼饼。是真的鱼虾做的鱼饼，还能吐出虾壳的程度。

奶茶店因为物价水平所以特别便宜，同行的朋友点了好几次霸王茶姬，我不喜欢喝奶茶所以没喝。

特产 #

砂拉越是咖啡产区，没想到吧！婆罗洲岛上的山脊，适合咖啡生长，因为咖啡树就是喜欢长在低纬度高海拔的地方。东南亚传统上喝咖啡喜欢喝罗布斯塔和liberica，这是两个咖啡树的品种，一般被拿来做速溶咖啡，就是一种不太讲究的豆子。然后罗布斯塔还会被烘得非常深，黑乎乎的，特别苦，没什么风味了，茶档里就用这种咖啡粉浓浓地煮一锅，便宜醒脑，功用上像那种叶杆子俨茶。喝不了这么黑的？这边白咖啡不是加牛奶的，加的是罐装的炼乳和 evaporated milk, 应该是因为天气热，鲜牛奶放不了。然后这一杯就甜甜浓浓，加进大半杯冰块摇摇晃晃，又提神又胖人。

但是随着精品咖啡的流行，砂拉越也开始种更多风味的阿拉比卡，也有更多烘豆商追求更精细地处理 liberica。我旅游也喜欢喝当地的咖啡店，现在城市都会有一些主打本地特色的。我虽然喜欢喝咖啡，但是都喝个乐子，没什么研究，也尝不出太多风味。买了一包本地种植烘焙的阿拉比卡回来当作特产尝尝，结果很一般！感觉风味很扁，偏苦一些，烘得可能比我平常爱买的要深一些，整个的口感也很扁平，总的来说就是感觉没有油！原产地的烘豆商经常吸引我，但是最近买的云南豆也是这个感觉，没有在处理和烘焙中 develop 好。

除了咖啡，砂拉越还产胡椒。在特产店里卖的多的是白胡椒，可能因为东南亚附近的烧菜习惯里还是不太用黑胡椒。黑胡椒和白胡椒是同一种植物大家知道吧？它也是一个果子，还不是太成熟的时候是黑的，不是，是绿的，绿的果子摘下来日晒完了就是黑的。等它多长一些再摘颜色就会褪一些，红红的。等到完全成熟，辛辣味散去一些，再水洗除去外皮，就会得到白胡椒！

整粒的白胡椒拿来煮汤和焯水去腥都很好用！

自然风光 #

砂拉越有一些溶洞，很多人爱去洞穴潜水。我们没有去，因为远，还要再坐车。下次还有机会的话我还挺想去溶洞划kayak, 感觉会好玩。

在城区往北开车一个小时的地方，沿海的热带雨林里，有个雨林公园，里面也有介绍一些原住民的传统建筑和风俗。这里每年会举办雨林世界音乐音乐节，灯光音响都挺破坏雨林的吧不是我说！在音乐节期间游客会多一些，对世界音乐感兴趣的朋友可以看看。

看看热带长得很狂野的植物，让人觉得植物也是一种有攻击性的活物：

出行 #

如果看到这里真的有人感兴趣想去玩的话，可以坐飞机到古晋机场，这个机场很小，非常小，过海关也挺快的。进城可以打网约车，这个软件叫 grab, 不绑定银行卡也可以用，付现金给司机就好。城里很小，随便走走路就打几个来回了。稍远一点的地方也都可以打 grab, 反正打车也挺便宜。
