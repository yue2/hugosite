分享一下本人作为科研工作者的爱用好物！对科研工作帮助不大，对生产力也没啥提高，最有用的就是一个买买买的爽感。以下有些是自己买的，有些是组里经费购买的耗材，不具体区分了。

那么就按使用场景来盘点吧！

办公室 #

键盘 #

人还是要有把机械键盘。其实没有也行，这种消费主义话术拿出来骗别人可以，别自己都晃进去了。我的工作是要做实验的，在办公室的时候就是看文献、写作业、和摸鱼这三项，其中摸鱼的时间最多，那么这把键盘最主要的作用其实还是摸鱼爽。

我在没有键盘的时候用笔记本电脑的薄膜键盘，就喜欢键程长一些的，比如说 thinkpad. 所以换成机械键盘完全能适应，无痛切换。手上这把是 ikbc C200 87键，茶轴，复古键帽配色。简单来说就是能买到的最便宜的机械键盘，对象给我买的（傍富婆文学警告⚠）。用着还可以吧，我没有更高追求，这个颜色我也挺喜欢的，就是米白色和灰色，营造一个上世纪电脑键盘的感觉，另外esc和空格大键可以换颜色，可以有一点点小个性，非常符合我小闷骚的审美，就是经过精心挑选，选择了看起来跟默认配色没啥区别的东西，但你要仔细问我，那我也是有理由的。

搜一个网络图：

87键基本上也和笔记本电脑的布局差不多，没有什么学习曲线。我用不上数字小键盘，而且办公桌就这么大，不想摊得太长。

鼠标 #

鼠标是最近买的，来自anker的竖款鼠标。我都不知道 anker 还做鼠标呢？！事实证明水平一般！

功能上在大拇指处有上下翻页物理按键，还有一个可以三档切换DPI的功能，用起来还可以，我自己的评价是：

但是这个竖着的鼠标是有用的 横着的普通鼠标画图久了会觉得拇指根这块肌肉很紧，肩膀也会酸 竖着的确实好一点 按键位置也还好没有需要特别适应

当时想买鼠标就是因为要画图，笔记本电脑的触摸板还是没那么好用，因为画图软件里很多功能是为鼠标键盘滚轮组合设计的，我之前在办公室都用台式电脑送的默认接线小鼠标，太小又太扁，确实很不舒服，也不方便在笔记本电脑和台式机之间切换。

现在这款是用usb接收器的鼠标，便携度好一些，也不用担心破台机的蓝牙能不能接上。缺点是还得占一个usb口。

显示器 #

显示器我挑了很久很久！因为工作上会做很多美工，业余爱好我也会拍照和小视频，所以想要一个色彩校正过的显示器。

然后我又不想要太大的，因为桌子小，24寸正合适。最后入选的幸运选手是 Asus ProArt这个24寸显示器，它还是16：10的横款比，更在我心目中加分！这个也是富婆买的，排队谢富婆请走这里——

缺点是分辨率 2k 而已，刷新率60 Hz而已，自带的扬声器垃圾得难以形容。但这些都不是我非常在意的方面，24寸的显示器2K看起来不寒碜，我也不打动作类的游戏。显示器现在有好多细分产品，各种规格组合，还是比较容易找到想要的款式。但是我这款可能不是大多数人的选择，所以不太打折。

耳机 #

办公室耳机我用有线的头戴耳机，用处是开视频会、听歌、摸鱼看视频，以及套着假装在忙，让别人不要来跟我讲话。满足以上所有条件的选择是铁三角 M20x 监听耳机。这是旧款，人民币大概三百块，用起来绰绰有余。配套买了个夹在桌子边上挂耳机的小架子，就这种东西：

这东西很好啊，不用把大耳机扔在桌面上。亚马逊上多得很，随便买一个就是了。

台灯 #

台灯也是今年买的，是从这儿看来的：

我买的这个是淘宝的便宜牌子，但也不是非常便宜，也要两百块人民币的样子。就是这样三折的，有两条都有灯。具体数据，就是流明和色温调整区间我忘了，还配了一个遥控器，但我就在办公桌上，所以没用遥控器。色温和亮度都可以离散或者连续地调整，这个功能很不错。

数据存储装备 #

数据存储太重要了！生死一念间啊！

U盘 #

U盘需要便携拷数据，从仪器到电脑、从电脑到打印机、从自己电脑到教室电脑都用得上，所以买个便宜大碗、小巧便捷的就好了。我买的是 sanDisk 32GB，这只小小的，我现在挂在门禁卡套上，真正便携。十块钱之内就能买到，人民币就是五十块左右。

我还有一只 sanDisk 32GB 双头，一边 usb 一边 usb c，也很方便，c type 这头可以用在手机平板和笔记本电脑上。

移动硬盘 #

重要的东西，比如说文章草稿，我会做三份备份，这是我师兄教的。就是电脑本地一份、网盘一份、移动硬盘上再有一份。

我十年前买的第一个移动硬盘，希捷 slim 500GB 至今还在服役，读写速度比不上现在的产品了，但是还是很可靠的。我还有一只西数 passport 1TB，足够我工作和业余爱好用了。

非常不推荐东芝的移动硬盘，读写的时候噪音贼大。

光驱 #

怎么搞的啊都要2024年了还要光驱啊！ 我问天问大地啊！

主要是有的仪器因为各种原因，比如说控制的软件很旧了，于是控制的电脑还在 win xp, 这电脑说什么也不敢接入互联网，又怕病毒（天呐！说出来都觉得上世纪！）所以不让用u盘，只能带DVD去拷数据。拷完回来我哪里还有能读DVD的电脑啊！

所以买了个外置光驱。这东西能用可靠就行，买了个可以读写的，一百块人民币的样子。因为实在忍不了了所以自己买的，没走办公用品报销。这是本人第一次为科研花自己的钱啊！

实验室用 #

iPad mini #

去年就吹过了这只 ipad mini，今年在前两天（好像已经是前两周）的电子手账文儿里也提过了，但是确实这就是我的实验室笔记主力产品，没带ipad就要用手机了。复制粘贴一遍我自己说过的话：

在生产力方面，我会带这个ipad去实验室，随时记笔记，表现很优秀，手写配合拍照，还可以进cleanroom（普通纸的笔记本不可以）。在MKBHD出的这款测评里，他说很多人不知道 ipad mini 在飞行员中非常热门，几乎是标配，因为只有这个尺寸能进cockpit. 我此处就想说 cleanroom compatible 也是很多人不知道的有用的feature! 这个尺寸比 ultra pro max 那种手机大不了太多，双手拿着打字回邮件也不麻烦。

炒这么多冷饭，我都感觉自己向真的测评博主更近一步了呢！

我的实验工作大概有一半时间是要在超净间里的，超净间就是没有灰尘的实验室，按照一个立方米体积中有多少颗粒，我们对超净间有相应的评级。那么任何会产生碎屑颗粒的东西都要避免，进超净间要穿防护服，因为人有皮屑和头皮屑；不能用铅笔；普通的纸也不可以；普通的纸巾也不可以。一般会发超净间用的笔记本，写字都要用圆珠笔。

iPad当然也不会掉渣，所以可以带。我同事用大号的ipad pro, 我觉得上课写笔记方便，拿来工作还是太笨重了，我还是选 mini！

耳机 #

实验室里我用主动降噪的，因为噪音就是很大，背景里有空调、空气过滤装置、冷却设备等等各种，没有耳机特别难受。

现役的是索尼 WI 什么什么 2 ，WI1000-XM2，对，挂脖子这款。索尼现在都不出挂脖子这款了我很伤心，因为我实在不喜欢真无线的降噪豆，取下来还要放盒子里，那这个盒子我还得到处揣着。而且因为项圈这款体积要大一些，所以降噪效果就是会比降噪豆好一些。这还有线控！我就是更信任实体按钮！这又不像大耳机戴久了闷得慌，而且我会戴耳钉，有时候大耳机角度没调整好会压着特别不舒服。

但这款评价比较差，我一人血书也难以拯救它的宿命。接下来我或许试试骨传导。

工具盒 #

咱工科的朋友们基本上人手一个工具盒，也可以有好几个，但我现在就一个。有人喜欢用那种大的电工箱比如这个：

我用不上这么大的。就在Daiso买了个小的塑料收纳盒子，类似这样

两层的就行了。上层放小工具，下层放要带的样品。我必备的小工具是透明胶带、记号笔、金刚钻、刀片、镊子。

镊子 #

镊子！实验室生活最重要的工具！

我有三把：

这就是我的工具箱。这三把镊子一把平头的，头是硬塑料的，磨损了可以换；另外两把金属尖头的，最细这个头的是用来拿取非常小的东西的。金属都是防磁处理过的钛和不锈钢。

镊子的选择有很多，我听师兄说在咱这行看到用某些系列的镊子基本上可以猜出是哪几个实验室训练出来的。

文具 #

再简单看下文具用什么

纸制品 #

  • 本子我用灯塔点阵 A5，就是开会听讲座的时候随便记一下，有用的回来再收拾进 todo list 和 onenote, 基本上没啥用，主打一个记了就是听了。一本能用一年多吧。
  • 办公桌上还有一沓便签贴，想起来就抓一张，基本也不用。
  • 元素周期表可以在这里打印。这网站挺好的，它把好几个图层都做进了这个pdf里，可以选择自己需要的信息再打印。

笔 #

  • 自动铅笔来自施德楼，日本产。这支自动铅用了得有十年了，是有一年过生日朋友送我的。按下去咔哒咔哒的反馈很给力，全金属笔身，重量在手上也很舒适。
  • 水笔圆珠笔随便买买就是了，我喜欢斑马的按下去的。

标签机 #

本组共有财产，是一只卡西欧标签机，好像是卡西欧的旗舰款。这玩意儿老高级了，可以改字体，还可以选装饰图标。有用是有用，但是功能真的忒多，自己买了放家里用的人真的会有种连环杀人犯的气质。

活页夹 #

用两孔的活页夹来装订打印出来的东西，配合一个普普通通的打孔机就完全够用。大的活页夹是随便买的，用来收纳以前上课考试的材料。但是我还要再吹一遍国誉这个薄的两孔文件夹：

用来放随时要取用的材料特别方便！比如说这学期在上的课、最近开会要讨论的材料、SOP等等……到时候用完了再收进大活页夹里就是了。

心情安抚用品 #

Fidget cube, jellycat, 亚克力小牌子，啥花里胡哨有用没用的爱买就买点吧。这活儿虽然工资低，但是精神压力大啊！

花钱的时候是开心的，花完钱不会吃土，差不多就得了。
