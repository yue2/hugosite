分享一些给别人剪头发和给自己选鞋子的经验。

在家剪头发 #

我有个朋友（真的有这个朋友）从来没有去过理发店，小时候是妈妈姥姥剪头发，后来是大学室友剪头发，再后来是对象剪头发，总之一直是在家剪的。上一次我去拜访这位朋友，就被抓壮丁，给人家剪了一次头发。剪头发之前我学习了一下油管教程，但是我现在找不到当时看的那个了所以没办法贴链接了！但是请大家相信我，剪得还可以，至少朋友是满意的。

这位朋友是长发，而我自己一直留短发，一两个月就要去一次理发店。我虽然不会给自己剪头发，但是看得多了也能学到一些，而且每次理发师都会问“想怎么剪”，如果说不出想法，可就要被当成创作了啊！在我的理论指导下，对象在2020居家隔离期间给我剪过头，效果也还不错。

这就是我全部的理发知识储备！即将分享给大家！

需要道具 #

  1. 剪刀✂️，用过两次厨房剪，比较锋利，但是大只，很难精细修剪。梳子也是必不可少的，还要有发卡或者小橡皮筋用来固定一部分头发。
  2. 凳子，让被剪头发的人坐着。
  3. 足够放下凳子并让理发师站着操作的卫生间。
  4. 清扫工具，扫帚或吸尘器。

行了，就这样。衣服肯定会弄得到处都是碎头发，别想了，洗了吧。剪完的头发也是要清洗的，所以围裙有没有也不特别有所谓。

理解一颗头和上面的毛 #

头是球形的，盖在头上的毛从头顶开始长，然后覆盖这个球形，所以头发大致可以分成几个区。首先请看图1:

可以看到头发分成侧面两块和后面一大块。前面如果要剪刘海的话，应该也是单独的一块吧，但是这我就不会了。从耳边把两侧的头发捞起来，剩下的就是后面的。

图1这是在这个球面上的分区，但是很显然在径向上头发也是分里侧和外侧的，请看图2:

这个里侧和外侧的区分在哪里呢？就在后脑勺凹下去的这个地方：

手指从这个两侧穿过，捞起来的部分就是外侧的头发，剩下的就是内侧的。短发可能看得更清楚一点，就是这样：

对于短发发型来说，鬓角也需要单独处理，所以两侧的头发也会分成里外两个区域。

好了，我们已经会把一头毛分成几块了，接下来按照分区轮流剪就好了。在剪某一块头发的时候，用橡皮筋或者发卡固定其他几块，就不会弄乱了。

案例分析 #

我们从应该是最短的那部分头发开始剪，以此为基准，就不会越剪越短越剪越少一发不可收拾。那么从上一节的分析，不难想象，头发应该是里侧比外侧短，两边比后面短，这样外侧的头发垂下去就会让整个发尾差不多长，显得有层次不会太厚重。

也有一些时尚发型不按照这个原则来，比如说非常知名的 wendy 头，主打的就是狼尾，也就是里侧的发尾比较长，戳出来向外卷。是很可爱啦但是好像不是我这个技术可以料理的。请看人家理发师好心好意画的辅助线：

那么就开始剪 #

还记得前两节的知识吗？

  1. 把头发分成区，一部分一部分地剪。
  2. 里侧比外侧短一些，两边比后面短一些。
  3. 先剪完成之后应该更短的那一区。

所以，我们可以从两侧开始剪，然后剪后脑勺内侧的头发，最后是后脑勺外侧的头发。

理论知识讲完了，但是我们没有经历过美容美发训练的人，当然会在执行中遇到技术问题。

第一大难点就是左右如何一样长。

当然是用辅助道具。可以用小橡皮筋或者发卡将两侧发尾扎起来，只留下发尾要剪掉的长度，可以调整橡皮筋确保两侧留下的一样长，下剪子时就剪到这里，两边就能差不多一样长了。

第二个问题是为什么在家剪的仿佛狗啃，没有理发师剪出来的自然？

这是因为我们按照分区一段一段剪了，但是每一段之间没有做过渡，看起来就会有点崎岖。要解决这个问题，操作也很简单。把剪刀这样竖起来，以比较垂直的角度剪发尾长度参差的部分：

多修几次，把台阶修成斜坡就好了。

推子简易技巧 #

突然想起来我还用过电动推子！用来给头发更短的同学理发了。

如果你愿意投资一点，也可以买一个推子。买来会获得这样一套产品：

可以把限位梳装在推子头上，就可以控制推出来的都是相同的长度。比如说装上3mm的，一条推下去，剩下的头发就都是3mm长。如果不装限位梳，那就是贴着头皮只剩个毛茬子的长度。

是不是很简单呢？

虽然非常简单好控制，但是用推子要做出比较好看的发型也是有技巧的，我看完教程就记得了一条，那就是不能满头都是同一个厚度的，不能装上一个限位梳，满头走一遍就call it a day! 为了这个造型，我们还是要做一点渐变。下面举个例子：

樱木花道这个头，在情节里是他自己推出来的，所以理论上应该是全头同一个长度，但是显然井上雄彦给樱木美容美发了，头顶的长度要长一些，这样才会显得脸型比较长啊！不然就是毫无特色的一个球啊！这样理发也不是不可以，主要靠的就是头型和一身正气了。

看下樱木的原型丹尼斯罗德曼：

没有进行渐变操作，直接推出来的就是这样的。

我们再参考长度渐变更明显的宫城良田。宫城的鬓角最短的地方应该是几乎贴着头皮推的，然后换稍长一些的梳子，从两侧一直推到顶，过渡到头顶卷毛。

总之用推子的要义是渐变，就算是寸头，也是从头顶一寸过渡到鬓角0.2寸，这样比较好看。

挑选适合走路的鞋子 #

我之前发现自己有一点足弓塌陷、拇趾外翻，给孩子吓得，到处搜索了一番应该怎么办。

从网络冲浪和看电视得到的结论是，要加强脚底肌肉的力量。

如果我们光脚站着可以发现，为了撑出足弓，大脚趾应该努力向中间张开。所以平常走路也要保持这个姿势的话，就要求鞋楦宽到有足够空间让大脚趾撑开。另一方面，鞋底应该尽可能地软而平，让脚自己完成这个走路的锻炼。

现在能买到的鞋，大多数太硬，鞋头太窄，所以大脚趾不可避免地受到挤压，不能撑开，久而久之就会拇趾外翻。然后又有的鞋子为了弥补这一点，做了一个足弓支撑的部分，这样虽然能让走路姿势保持稳定，避免脚踝和膝盖遭受连环反应，但是反而让脚底肌肉更不用锻炼了啊！

经过这么一通分析，不难发现，生物力学上最合适的鞋就是——锵锵锵锵——人字拖！

可惜我不是老广包租公包租婆呀，不能天天穿人字拖，我还得上班呢。所以还得找个满足要求的鞋。

风很大的裸足鞋 #

在学习完这一套理论之后，我就发现了潮流时尚的裸足鞋。基本上是按照上面提到的这些原则设计的，适合走路。

在阅读了相当一些网上测评之后，我挑选了splay这个牌子，因为它有非常普通的帆布鞋，是我会日常穿着的款式。这个牌子提供的尺码表有一个可以下载打印的pdf, 这个很好用！可以相当自信地网购！这一点也让我对这个牌子很有好感。

第一次穿上这个鞋的时候我感到非常震撼！原来鞋头是要这——么宽的吗？一下子感受不到任何来自鞋的挤压，还有点没有安全感。但是穿了两次就习惯了，果然是这样比较舒服！可以很直观地感觉到，自从换成了裸足鞋，我甲沟炎的苗头都没有了，原来之前是压的！

裸足鞋是对脚比较好没错，但是它因为追求这个裸，所以也是没有任何保护的，我目前只是用来走平路和举铁，跑步需要缓冲、打球需要缓冲和抗扭，这些场合还是会换成其他功能的鞋。

别的牌子还是可以继续买 #

裸足鞋虽好，但是相比起它普普通通的质量、平平无奇的帆布、完全不修饰的外形，这个价格还是贵了点。牌子又都比较小，不怎么打折。

所以在体验过裸足鞋之后，我觉得自己能知道脚趾应该是什么感受了，就又回到了更常见的牌子！买了这两双：

NB 574 是知名的宽头款，穿起来是比较舒服的，而且NB提供半码，在犹豫不决的时候总是可以选大半码的鞋，穿起来更宽松。小缺点是这个底不是完全平，比较厚而且有一个小坡度，并不完全符合裸足的要求，也不适合穿着举铁。但是可爱啊！穿上574更觉得自己是中学生了！

另一款是 adidas samba 德训鞋，看起来鞋型是窄的，但是因为皮很软，所以完全没有不舒服，脚趾活动空间绰绰有余。这款的底很平也很软，它这么流行果然不是没有道理啊！

买鞋的要点是活在当下 #

总之我发现，买鞋不能太想着虽然现在有点挤但穿过几次就合脚了，重要的是现在就合脚！现在就舒服！现在挤脚，那就是现在被挤了脚。 在和鞋的磨合过程中，拿来磨合的可都是我自己的脚啊！
