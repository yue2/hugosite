 Middle of Nowhere

博客作者问卷

2024-10-23 *edited 2024-10-23 — 7 min read
#blog 

我最喜欢做网上问卷了！

原问卷是来自@dayu。受 @杜尚在此沙龙邀请，却之不恭，本宝也来写一写！正好有一些起源类的问题时不时会被提起，有时候是有人问，有时候我自己憋不住想说，那就一起说一说吧，所以可能会借题发挥多写一点。

1. 简单介绍下自己或者你的博客？ #

我叫陆博学，关于页面在这里，但是写得很简略，约等于没说，不如此次多说点。

这当然不是我的本名，但是也相差不远，朋友同学之间本来就是叫我阿学的。我之前在书店闲逛，看到现在出版的作者多以网名行于世，花里胡哨的，我就想，我偏要一个有名有姓正正经经、特别像是真有这么个人的网名/笔名，譬如鲁迅（碰瓷呢）。我平常爱看侦探、间谍小说作为消遣，不知从哪本书里得来，化名应要或多或少与本名相关，不至于在街上被叫了化名，一时之间反应不过来。于是便定下了还是要叫阿学。可惜后来又看了退休cia间谍的采访，人家说这是小说里瞎说的，改头换面的要义当然是没有一处和本身相同，别人日后回忆起来，应该所有情报都是假的。拜我学艺不精所赐，伪装术不到位，但这名字已经定下，便是如此了。

但这样也好，和网友交往起来也和三次元朋友相似，反正都叫我阿学。我少时家教严肃，很有安全意识，保护欲过头，我自己也没什么网上爱好，所以从没交过网友，相处的都是同学和兴趣班的师兄弟姐妹。现在终于有机会结识网友，但是我和人交往的习惯早就养成，所以也还是把网友当作一般朋友相待。

这个博客的名字叫 middle of nowhere, 前不着村后不着店，一方面是我确实不认路，经常坐错公车把自己搞到前不着村后不着店的地步；另一方面我在人生上是这个状态，感觉总在一个暂时的状态，不知走到何处了。我评价起自己来，总是有一点悲观否定的。

如果要乐观一点地劝勉自己，那即便是前不着村后不着店，今时今日也还是要过的，过着过着，不也写出了这小一百篇篇博客？——突然扣题！

2. 什么契机让你开始写博客？ #

没有契机啊，我本来就在写，我中学就在写QQ空间，长文章，八百一千字那样的，写些日常故事、看的电影、听的歌，后来上大学兼任一些校内时政评论，也就是骂老师骂辅导员骂（副）校长，还挺多同学看的。但QQ上都是认识的同学在看，把这些作为博客搬出来给网友和更多不认识的人看是2020年的事儿，pandemic 居家嘛，人一无聊胆子就大了。

刚开始在豆瓣放了两篇，立刻就被豆瓣的日志审核烦的要死，拔腿就跑。另一方面我在豆瓣人微言轻（嗯？），豆瓣上的网友又给我一种印象动不动就会嘲讽别人，有攻击性，我没有安全感，不敢久留。现在这个博客我连评论区都没做，眼不见为净！背地里怎么骂我随你便！

3. 你是如何完成创作的？ #

技术上，后端的文字编辑软件是用的 obsidian，代码编辑软件原本是 atom，现在 atom 被 github 停了，于是刚换成 vs code, 但是懒得搞，仍旧把插件和页面排布做成 atom 那样莞莞类卿地用。这个库我自己在 onedrive 上同步备份着。

obsidian 还不错，光拿来写 markdown 属于屈才，但是屈才的这部分做得也挺好。显示样式可以自定义，我喜欢黑的菜单栏配白的页面。另一个好处是可以用 drop down menu 选择 front matter, 也就是分类和标签那些，用起来很自然。不过我一开始是直接在 atom 里写的，谁跟它比都挺自然的！

平常的片段灵感是用 simplenote 在写，这个软件好，我喜欢，好就好在网页版也是一样地用，另加上 ui 简洁、全平台、同步快，我有任何大事小事都在这上面记。也有时候直接想到个题目就写在todo 软件上，我用的是 tick tick, 理由也是一样的，网页版、ui 简洁、全平台、同步快。我喜欢的工具都是上手极快非常直观的，我很讨厌一个工具它的宣传点竟然是熬过这陡峭的学习曲线就是一方境界。我这日常工作已经够有学习曲线了！买个螺丝刀还不能直接用是为什么！

至于写什么、怎么写的博客之道，我写过一个回答，简而言之不过就是爱写什么写什么，创作是一种消遣放松，以自己舒服为要。

4. 运营博客的过程中是否有失去过动力？如果有，是为什么恢复的？如果没有，请问您又是如何保持创作的激情？ #

没有。如上所述，我觉得写博客是一种消遣放松，不算什么激情所至，更无所谓“运营”，说实话我根本不知道谁在看。想起来就写了，这一阵子没有动力不想写就放着，过一阵子又想写了。只要把时间尺度拉长，就不算放弃，就永远在坚持。话虽如此，我理解有的时候也会有一蹶不振的感觉，所以当然也不能拖太久，几个月里总要写个什么。拖到笔端驽钝（键盘卡住）之时，就找个特别容易写的题目速速糊弄一篇出来，然后有了这个 momentum, 一切又能运转起来。比如这样的网络问卷就是很容易上手的题目！

像我这样完全不花心思“运营”博客的，有时候可能会觉得有点寂寞，也想和人交流评论，但更多时候是对加入这个“博客写作者社群”的不安，所以我也不敢以此自居。我当然从小就是这样，踌躇徘徊，没有参加过任何圈子。我也会觉得很遗憾，但是一直如此，便不知道要怎么开启一段社交，自己过自己的也就算了。

5. 如何搭建博客，以及运营博客每年需要投入的资金？ #

这是一个静态网站，用的是静态网站生成引擎 hugo, 托管在 github 上，由 vercel 部署，这几样东西都不要钱。这样做需要一点 html, css, 和命令行的代码知识，挺简单的，我折腾过好几回，大不了炸了全都重来，完全不怵。如果有朋友现在想上手就更容易了，可以直接 chatGPT 问代码。这个域名一年十几块，就是我全部的成本了。

我还主持一个播客节目，有个节目的网站在 wordpress.com 上，是免费档，所以也没花钱。背靠大公司的好处当然是人家有专人运营，连接稳定还能抵御基本的网络安全问题。节目网站不好随便炸，所以放在上面放心一点。而且那是需要和人互动的，用动态的 wordpress 更合适。

6. 推荐 1 篇你博客中的文章，并推荐一个你喜欢读的博客，聊聊原因。 #

我不知道自己有什么好推荐的，好像我的旅游文章比较受欢迎，就放这篇《坐火车去越南》吧。这个系列的其他几篇也挺有意思的，写得随随便便乱七八糟的，并不具有旅行参考价值，也没多少文化历史收获，就只是写着玩儿看着玩儿的，读起来轻松。有网友夸我写东西简洁明快，风格像小朋友，谢谢网友抬爱了。我有时候会写一些很自我剖白的随笔，想到会被不知道什么人看还有些不安，感觉未免太暴露，所以也不太分享出来，当然我既然放出来了那也不拦着，有缘看到就看到了。

我有很多喜欢的博客！提两个我感觉很低调的给大家认识认识吧： Samantha不常写但是写很好！我很爱看人家对生活能有这么深入的反思，表达水平也好。而且 sam 的业余爱好似乎是谈恋爱（对不起），看八卦还是很好玩的（对不起）。阿圆的博客更技术一些，但是阿圆不是那种技术 snob, 所以完全不会有那种让人捏鼻子的讨厌技术宅气味，更像是爱折腾，所以看起来很有感染力——这么能折腾，真有精神呢（对不起）。而且阿圆的审美偏好深得我心，网站做得简洁漂亮又闷骚（对不起）。

7. 推荐 1 个近期喜欢的事物？ #

我还有个和博客搭档一起做的月更 newsletter，会写最近喜欢的东西，也就是一些文化消费和口腹之欲。除此之外我自己最近爱用的就是这台 macbook air m3，趁手工具，什么都好，上手也快，缺点是太贵。推荐说不上，这么贵的东西，不敢随便推荐给别人。本来说要单写一篇讲讲我第一次换苹果电脑的体验来着，欠着吧，下次一定。

8. 想做还没有做的事，或想尝试还没有尝试的主题？ #

我一直是评论者和记录者，一直想试试创造，也就是写虚构的故事，但是确实不会，也没搞过同人，没有那个勇气。

另外还有很多装修的事情想做，要改漂亮点的高亮和链接样式、要做 call out banner, 要做可隐藏段落、想改个时髦一点的字体、看人家的 breadcrumb 做得也很简洁大方想抄一下……不一而足，下次一定！

9. 写到这里，闭上你的眼睛，深呼吸几分钟，或是出去溜达一圈，然后回来写任何你想写的东西。 #

溜达一圈买了个热狗小面包，还温着呢，松松软软香香甜甜。
