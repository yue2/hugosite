等我反应过来，2024的第一个月已经过完一半了，赶紧得写个新年展望，不然都不好意思说是“新年”了。

指导思想 #

我希望2024年过得 intuitive and mindful, 就是说还是根据直觉来做事，饿了就吃、困了就睡、累了就休息一下，但是我同时也希望对自己在做什么想什么比较有数，不要毫无知觉地让一切从光滑的大脑皮层流过。

这是总体指导方针，具体想要做得更好的一个方面就是践行环保。

我知道以我的收入和消费水平，对地球的破坏力其实很有限，游艇、私人飞机、牛奶浴、吃夏威夷果长大的和牛做的牛排，哪个都是我要不起的，我的生活方式本来对地球的负担也没这么大。那我更环保一点对地球的帮助也没有很大？——确实是这样。但是这不代表没有帮助，不代表我就可以不帮助地球了。如果以我自己作为一个平均水平，所有人都做一点，那么对减排的影响就很大了。而且另一方面，做了一点是一点，多做一点总比少做一点好。

站上这个道德高地，接下来我还得承认，我对环保事业也没有特别投入，主要就是在可以选择环保的时候就尽量选更环保的方式，所以我说这是“凑活环保”主义。

已经在做 #

在我现在的生活方式里，凑活凑活已经做到的也不少。

比如说第一条就是买星巴克自己带杯子。这很容易做到，因为我常买的是我们工学院楼里的那家星巴克，我就从app上下单，然后从办公室带着杯子去打回来。星巴克也会为此给我便宜几毛钱，何乐不为！但我不是每次都能做到自带杯，有时候甚至会因为懒得洗杯子多花这五毛钱。新的一年就决定从这里开始动刀，目标是在这家店买东西每次都自带杯！这样的目标就是很小很小的一个进步，对我的生活方式没有太大的改变，总的来说还是 intuitive 的，想喝星巴克的时候就去买，但是同时更 mindful 一点——记得每次都带杯子。

我去超市买蔬果也不会扯塑料袋来称重。很简单啊，就直接放上去称重，然后打印出来的条码直接贴在瓜果皮上，反正很容易撕掉，吃的时候又不会吃这个部分。如果是不容易贴和撕标签的蔬菜，就把这个条形码贴到另一个橙子皮上好了，并不会混起来。

自从我把迪卡侬买的环保袋扔在书包里每天带着之后，去超市就几乎不用塑料袋了。因为经常有时候下班顺路去买点东西，而买菜帆布包在家里，于是又要拿超市塑料袋。而迪卡侬这个是那种可以折叠成小方块的环保袋，叠起来放包里不占地方，随时拿出来都能用。解决方案竟然如此简单！环保袋竟然真的实用！本地现在有些高级超市直接不提供塑料袋了，只有卖好几块钱的环保袋，我觉得这样也不好，真的忘记带买菜包的时候又要花几块钱买那么结实一个环保袋，搞到最后家里好多环保袋。我很怀疑这样做是为了环保还是为了增加营业额。我就有一次去这个超市买东西忘记带买菜包，到收银台前傻眼了，最后以一个托塔李天王的姿势带着一颗菠萝坐了公车。

我还不喜欢吃外卖。这也不纯粹是环保的原因，主要是外卖的味道就是没有店里的好吃，当然啦，很少有食物能出锅后等半个小时还一样好吃。而且外卖费很贵，算上配送费、平台服务费、还有外卖软件菜单偷偷比店里贵一点，太不划算了！不吃！！还有吃完之后外卖盒在家里垃圾桶里散发出将馊不馊的味道我特别讨厌，所以在2020之前几乎从来没有吃过外卖。在家封锁期间吃得比较多，现在又不怎么吃了。

怎么会有人写 new year resolution 上来先把自己夸了一通啊？！

要做什么 #

好了接下来真的开始讲要做的部分。

别买瓶装茶 #

首先第一条就是把工作日常买的瓶装茶戒了。买的真的太多了！几乎每天都在买啊！！开始我的忏悔：每天吃完中饭，哎呀食堂好油——买瓶茶刮刮油；今天晚上还要工作一会儿——喝瓶饮料提提精神；想喝点有味道的东西——那就买瓶椰子水。总之真的是太多了。而且我知道自己买太多瓶装饮料，每次扔塑料瓶时都隐隐约约有点歉疚，但还是在买啊！

决定把冷萃瓶和茶包带去办公室的茶水间，做自己的茶饮料。本来就买了好多茶包茶叶不经常喝，带去办公室让它们物有所值也好。然后便利店瓶装饮料的份额就控制在每周一瓶吧，允许自己忍不住了还是买一瓶。

我会记录一下Q1的表现，如果发现也能忍，那就可以进一步减少至一个月两瓶。

记录消费情况 #

有一个 3R 的环保原则大家应该都听过，就是 reduce, reuse, recycle. 这个是有顺序的，最有用的就是 reduce ——不买即省百分百。服装制造业对环境的影响是很大的，少一件衣服被制造出来就是少给地球造成一部分影响。

我自认为是很少买衣服的人，因为我对这方面不是特别感兴趣，工作环境又可以穿得很休闲，所以很少有“逛街买衣服”的印象。但是！拍脑袋的印象要不得啊！我怀疑自己每次都说着哎呀难得一次，然后就买了，然后难得着难得着，其实买了很多。

最近难得的产品是为了过年买的红色袜子😅

所以我打算更精确地记录自己衣柜入荷情况，看看我到底买了多少！目标就定到这里，先看看，对消费情况有数之后再作下一步打算。先计划个半年吧，我们Q2再来回顾一下。

make 手帕 great again #

我一直到上中学都是用手帕的，因为觉得可爱，买过几方特别可爱的手帕。后来总是不记得带，就慢慢放弃了这个习惯，被抽纸的便捷迷惑了！新的一年，让我们 make 手帕 great again! 重新买一两块可爱的手帕，擦嘴可以用、桌子上撒到水也可以用、吃水果擦手也可以用。

吃更多素菜 #

吃饭习惯上，我还是吃肉的。我有几个朋友就是出于健康和环保的考量，开始吃素食。我不打算现在就 commit 这么多，所以饮食习惯上我想只是多吃一些素菜。具体来说，我把目标定在每周吃两顿素菜，而且是蛋奶素，因为我还是无法割舍鸡蛋！

其实我本来的饮食习惯也不是非常不环保，首先因为我吃牛肉很少。我也不是不爱吃牛肉，主要是好吃的店并不多。这是我的一个消费原则吧，既然不经常吃，那每次去吃的时候都吃好一点的。好钢用在刀刃上！牛肉吃得少，我平常吃得最多的肉是鸡肉，辅以猪肉，另外我还很喜欢吃鱼虾，所以这个饮食结构的碳排放没有这么大。然而相比之下，吃植物类的食物的碳排放还会更显著降低，所以我想多吃一些蔬菜。

我希望能找到几家爱吃的素食店和可以经常点也不会吃腻的素菜单品。食堂的素食窗口我也吃过几次，但口味不是很喜欢。我经常吃的沙拉店有一个 impossible patties, 就是用植物肉做的肉饼，味道很不错，肉饼里加了香菇，调味像是小时候爱吃的桂冠香菇贡丸。煎到两面上色，香香！我还喜欢吃一切鹰嘴豆制品，hummus, falafel, 罐头鹰嘴豆直接吃，干脆鹰嘴豆夹心的巧克力小零食，都很好吃。这些爱吃的素食我应该多给它们出场机会，而且我也想主动去探索一些其他的素食菜肴。

除此之外，我计划在新年里掌握两个拿手素菜，是把蔬菜作为主角的大菜！我现在烧菜一般是有一套公式，即一个肉菜、一个蔬菜、一份碳水，这样可以大致满足蛋白质、纤维素、和碳水的平衡摄入，而且计算菜量我也很习惯了，买菜烧菜都很顺手。目前这个情况下，我一般只起一个油锅，用来做这个肉菜，调味也会比较重，然后蔬菜通常是烫一下或者凉菜的形式。关于组织一顿饭的心得下次单起一篇吧（下次一定），对于这次的新年计划来说，就是想熟练掌握两个需要起油锅的、滋味很丰富的、蔬菜料理，学习以蔬菜为中心元素来组织这顿饭。

目前还要继续浪费的 #

棉条 #

卫生用品上，我还是在用棉条。我观望硅胶制的可重复使用的产品很久了，但是因为还处在经常搬家的租房状态，我对于洗消存放硅胶制品还是有点不放心，所以依然不打算换。对不住了，还要继续用棉条。换用可重复使用的生理期产品会持续在我的计划中，希望可以早日下定决心！

又开始买纸质本子了 #

我已经无纸化工作很多年了，主要也是因为经常搬家，书和本子都很麻烦，而且电子化的内容方便搜索，对于工作来说非常必要。办公室的一包A4纸已经两三年了我都没用完。

但是最近心情上想重回青少年时期，又回到了见到可爱小本子就想买的阶段，所以买了好几本纸质本子。而且不可否认，买到漂亮小本子真是好开心。可得好好用啊，树都砍了，还放着不用就太不像话了！
