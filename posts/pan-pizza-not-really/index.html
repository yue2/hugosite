Pan pizza (not really)
2021-05-15 16:22 folder 生活   label 菜谱   记录  

小高姐菜谱 https://www.youtube.com/watch?v=ATEnM1YPQQE&t=178s

  1. 面团（2份）
    1. 中筋面粉：250克=1 2/3 杯
    2. 盐： 2克
    3. 油： 13 克 2
    4. 酵母：1克
    5. 室温水：175克
      • 40%含水量面团=175g水/425g总量
      • 用sourdough starter发酵可用总质量20%的酵头=85g，酵头含水量100%=42.5g面粉+42.5g水
      • => 酵头面团
        1. 207.5g中筋面粉
        2. 132.5g室温水
        3. 85g酵头
        4. 油13g
        5. 盐2g
  2. 面团制作
    1. 混合面粉、水、酵母（酵头）、盐，拌匀成团。面团很湿，勉强成坨没有面粉结块即可
    2. 盖上盖子休息1h，然后从边缘折起面团，折叠几次使面团较光滑
    3. （可选）1h后重复一次上一步
    4. 加入油包裹面团表面
      • 酵头发酵比较缓慢，所以前1-2h不要加油，让它自由发酵
    5. 盖上盖子冰箱4℃发酵24-48h
  3. 红酱
    1. 锅烧热，加拍扁的蒜头2-3个炒香
    2. 加入罐头番茄，用罐头接水冲干净番茄汁
    3. 加入罗勒梗
    4. 加少量盐，小火煮至略微收汁（比新开的番茄罐头稍粘稠一点）
  4. 烤
    1. 最高温预热，550F或300C。把铸铁锅放进烤箱一起预热
    2. 面团恢复到室温，在烤盘纸上撒干面粉，将面团转移整形
    3. 组装披萨
    4. 转移到铸铁锅，烤5-7min，最后烤炙2min至上色
  5. 用菜刀或披萨刀（如果有）切成块
  6. 吃
